# TiacTacToe
Simple game implementation with pygame. OOP. Two modes: PLAYERvsPLAYER and PLAYERvsCOMPTUER (minimax algorithm).

In order to change the game mode, you need to run different methods **(run_game, run_game_vs_ai)**
